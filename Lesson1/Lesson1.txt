Pierwszy system wudowany to system Apollo, sam reagowal na wydarzajace sie sytuacje

Def.  Jest to system komputerowy, zaprojektowany aby dzialal autonomicznie
      Jest to hardware lub software, jest oparty o mikrokontrolery lub mikroprocesory


PE -> Processing elements (jednostki obliczeniowe) -> wykonywanie zadan

CL -> Communication links -> magistrale komunikacyjne


PP -> Programable processor (programowalnie jednostki obliczeniowe) -> uniwersalne jednoski obliczeniowe ktróre wykonują obliczenia
HC -> Hardware cores (jednostki sprzętowe) (moduły dedukowane) -> Jednostki sprecyzowane do wykonywania jednej określonej czynności


Wartosci systemow wbudowanych:
    -> Pobór mocy
    -> Wielkość
    -> Koszt (koszt architektury, koszt wykonania zadań)
    -> Czas wykonania jednego zadania i wszystkich zadań
